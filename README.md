# 🧠 Informya

Informya is a personalized news application designed to help users understand the world through curated, multi-source information.

## ✨ Features

- Personalized news feed based on user interests
- Multi-source aggregation (RSS, APIs)
- Intelligent filtering to avoid duplicates
- Bias-aware content selection
- AI-assisted summaries and explanations

## 🏗️ Tech Stack

- Backend: FastAPI (Python)
- Scraper: Rust
- Mobile App: Flutter
- Database: PostgreSQL
- Cache: Redis
- Infrastructure: Docker

## 🎯 Goal

Build a modern, intelligent news experience that prioritizes clarity, diversity of sources, and user understanding.

## 📱 Platform

- iOS (Flutter)
- Android (planned)
